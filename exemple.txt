Text editing language : 

Language feature : 

  Go to left/right/down/up
  start/end of the line,
  start/end of the file,
  advance/back in word, 
  substitute,
  Insert/Delete string 
  Copy/Paste word ,
  Global function,

Language syntax : 
  
  It is highly inspired by vim, we use the following syntax : 

    Line number : COmmand 
    command times on what 
    Int Move command, 
  
Exemple of command possible 

  Possible token : 

    GLOBAL BOTTOM
    RIGHT 
    LEFT
    HOME
    END
    INSERT 
    DELETE
    COPY
    PASTE
    SUBTITUTE 
    MARKER
    GOTO

Exemple of command possible : 
  
    1 : 'copy line';  6 : 'paste,';  3 : 'delete 3 right, insert "bonjour", end , insert "chien"
    5 : '3 left, marker b'
    2 : 'subtitue "pattern" "replacement"'
    GLOBAL : 'copy 3 left , paste , end , paste ,  '

File contained in the final project :

mll for the lexer
